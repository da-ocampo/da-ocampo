About Me

Based in Houston, Texas, I'm a front-end developer, proficient in various programming languages and frameworks.

Throughout my career, I've worked as an independent developer, leveraging my self-starting mentality to deliver exceptional results. My interest in web development sparked during high school, where I volunteered to code my school's website.

This experience prompted me to enroll in the Associate of Applied Science in Web Development program at Wake Technical Community College immediately after high school. During this time, I honed my skills through group projects within the program and freelance work outside of it, creating small websites for local businesses and musicians.

Portfolio

ihateomerta.com

Omerta is an experimental five-piece band hailing from Houston, Texas. The website, built using HTML, CSS, and JavaScript, serves as a centralized hub for fans. It provides links to various streaming services and social media platforms, along with essential information like tour dates and blog posts from band members. Additionally, the site features an e-commerce store built using Shopify.

painproxy.com

Pain Proxy is a virtual venue built with HTML, CSS, and JavaScript, leveraging Three.js for 3D rendering. It serves as a platform for artists of different mediums to showcase their artwork. The space aims to foster creativity and provide a safe environment for artistic expression.

danielsgone.com (v1)

Daniel's Gone is an electronic music project based in Houston, Texas. The site is built using HTML, CSS, and JavaScript, serving as a hub for listeners by providing links to various popular streaming platforms and other ventures under the project's name. In addition to creating all front-end components, I also designed all website assets using Photoshop.
